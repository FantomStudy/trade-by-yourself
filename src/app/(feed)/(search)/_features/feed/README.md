# Feed Feature

## Ответственность

Фича **feed** отвечает за отображение ленты объявлений с возможностью фильтрации и сортировки.

## Core функции

1. **Компонент ленты продуктов**
   - Загружает список обьявлений
   - Фильтры передаются пропсами извне
   - Бесконечная подгрузка + кэш

2. **Поиск товара**
   - Используется search-params(с помощью nuqs) для удобного восстановления результатов поиска
   - Инпут поиска по тексту
   - Диалог с выбором категории
   - Диалог с выбором локации (в планах)
   - Виджет с фильтрами по полям типа подкатегории (в планах)
