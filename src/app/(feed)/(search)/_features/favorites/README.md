# Favorites Feature

## Ответственность

Фича **favorites** отвечает за управление избранными объявлениями пользователя.

## Core функции

1. **Добавление / удаление товара в избранное по кнопке**

   - Работает по кнопке, используются оптимистичные обновления

2. **Компонент списка избранного**

   - Подгружает список избранных товаров
